# HTML Audio & Video Player

A simple project built as part of the freeCodeCamp Responsive Web Design curriculum.

This project demonstrates how to use HTML5 media elements to embed and control audio and video content directly in a webpage.

## Features

* HTML5 video player
* HTML5 audio player
* Video poster image
* Playback controls
* Loop functionality
* Responsive media display

## Technologies Used

* HTML5

## Concepts Practiced

* `<video>` element
* `<audio>` element
* `<source>` element
* `controls` attribute
* `loop` attribute
* `poster` attribute
* Media file embedding

## Project Structure

```html
<video>
  <source src="video.mp4" type="video/mp4">
</video>

<audio src="audio.mp3" controls loop></audio>
```

## Learning Outcomes

Through this project, I learned:

* How to embed audio and video files in HTML
* How browser media controls work
* How to provide video thumbnails using the poster attribute
* The difference between media file URLs and webpage URLs
* Basic HTML5 multimedia best practices
